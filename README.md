# NonLethalFlares
Very simple mod that prevents flaregun projectiles from causing damage in The Long Dark. 

This has always been the strangest thing about bear encounters for me, to the point where I try to avoid them because it's just free meat - they always die eventually. 

With this mod, there is no bleeding or damage inflicted to the bear. It will run out of aggro range and go back to wandering.

REQUIREMENTS: ModSettings
